# Fault-Tolerant Federated Learning Framework

This project simulates a fault-tolerant federated learning system for handling client dropout.

## Features
- Reliability-based client selection
- Client dropout simulation
- Adaptive weighted aggregation
- MNIST image classification
- Accuracy and loss graphs

## Install
```bash
pip install -r requirements.txt
```

## Run
```bash
python main.py
```

## Output
Graphs will be saved inside the `results/` folder.
